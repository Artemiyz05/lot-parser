# Lot Parser: Инструмент для сбора и анализа аукционных данных
Lot Parser — это инструмент для автоматизированного извлечения данных с площадки «Торги России». Скрипт позволяет быстро собрать информацию о лотах, не прибегая к ручному поиску, что значительно ускоряет анализ рыночных предложений.
# Зачем и где применяется?
Программа разработана для случаев, когда требуется оперативно получить сводную таблицу по актуальным торгам.
- Избежать рутинного копирования данных из браузера.
- Подготовка отчетов, мониторинг новых поступлений в конкретных регионах, первичный сбор данных для аналитики.
# Что умеет и как работает
Проект реализует полный цикл сбора данных (Scraping Pipeline):
- Навигация: Автоматический обход страниц листинга (пагинация).
- Глубокий поиск: Переход в каждую карточку лота для извлечения подробных характеристик (Регион, Цена, Тип аукциона и др.).
- Обработка ошибок: Использование блоков try-except позволяет скрипту продолжать работу, даже если структура одной из страниц повреждена.
- Отслеживание состояния: Логирование всех действий в parser.log
# Анализ производительности и оптимизация
На текущий момент программа работает в синхронном режиме.
Важное примечание: 
Данная версия является стабильным MVP (Minimum Viable Product). При обработке очень больших объемов данных (тысячи запросов) скорость может быть ограниченной из-за последовательного выполнения задач.
Планы по развитию:
Для решения вопроса масштабируемости следующим этапом разработки станет внедрение асинхронных технологий (asyncio, aiohttp или httpx). Это позволит выполнять сетевые запросы параллельно, что ускорит работу скрипта в несколько раз.
# Технологический стек:
- Python 3.10+ — основа проекта.
- BeautifulSoup4 & lxml — для парсинга DOM-дерева.
- Requests — для работы с HTTP-запросами.
- Pytest — модульное тестирование логики.
- CSV — формат хранения данных для легкого импорта в Excel.

# Установка и запуск
1. Клонирование репозитория:
git clone https://github.com/Artemiyz05/lot-parser.git
cd lot-parser
2. Настройка виртуального окружения:
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
3. Установка зависимостей:
pip install -r requirements.txt

Запуск инструментов:
- Парсинг: python main.py
- Тесты: pytest test_parser.py
